# Frontend Design Guide

This project uses a simple, modern frontend design system for Vue 3 + Vite applications.

## Design principles

- Keep the interface clear and friendly.
- Prioritize readability over decoration.
- Make navigation and actions obvious.
- Ensure layouts work on mobile and desktop.
- Maintain good contrast and accessible interactions.

## Visual direction

- Use a dark, polished UI style for AI product experiences.
- Use a restrained color palette with one accent color.
- Keep spacing consistent and generous.
- Use rounded corners and subtle shadows for depth.

## Component guidance

- Use Tailwind for layout and spacing.
- Use Element Plus for common UI patterns such as buttons, cards, and dialogs.
- Prefer small reusable components over large monolithic views.

## Content guidance

- Use short headings and concise descriptive text.
- Put the main action early in the page.
- Show status or feedback clearly when something changes.

## Workflow tips

- Start with a simple layout.
- Add visual polish after the content structure is solid.
- Review the page on both small and large screens before finishing.
